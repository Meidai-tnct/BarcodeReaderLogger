# BarcodeReaderLogger
読み込んだバーコードを日時とともに保存する

## 使い方
- テキストボックスにカーソルを合わせてバーコードを読み込むと日付とともに、  
同じフォルダにテキスト形式で都度保存される。
- ファイル名はlog_(日付).csv（画面左下に表示）
